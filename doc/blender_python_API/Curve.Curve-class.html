getMode()

Get the Curve mode value. The mode of the curve is a combination of 4 parameters.
  • Bit 0 : "Back" is set
  • Bit 1 : "Front" is set
  • Bit 2 : "3D" is set
  • Bit 3 : "CurvePath" is set.
  • Bit 4 : "CurveFollow" is set.
